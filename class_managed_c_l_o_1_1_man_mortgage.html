Detailed Description

Class that describes a loan.

This class is used to describe a loan characteristics and to calculate the expected cash flows generated by the loan.

String^ ManagedCLO::ManMortgage::Annuity
getset

Vector defining if a loan is annuity or interest only.

Bloomberg-style vector defining whether the bond is an annuity.
The vector can assume the value N for interest only and Y for annuity and can only move in steps.
Examples of valid vectors:
Y
N 24S Y
Examples of invalid vectors:
5 24S N
N 24R Y

String^ ManagedCLO::ManMortgage::HaircutVector
getset

The Haircut.

This vector represents haircuts to apply on the loan in percentage points.
Examples:

  • 10 12S 20 will apply a 10% haircut on day 1 and a further 10% after one year.
  • 0 24R 50 will gradually haircut the loan up to a total of 50% of the original value.
  • 0 12S 30 12S 10 will cut by 30% the value of the loan after one year and then recover 20% on the second year

String^ ManagedCLO::ManMortgage::LossMultiplier
getset

The multiplier for the loss amount.

This multiplier scales the losses incurred on the loans.
The base is 100. Values greater than 100 denotes greater losses on the loan and viceversa.
The formula to calculate the loss will be Outstanding * CDR * LS * (Loss Multiplier/100)
Bloomber vectors are accepted

String^ ManagedCLO::ManMortgage::PrepayMultiplier

The multiplier for the prepayment amount.

This multiplier scales the prepayments of the loans.
The base is 100. Values greater than 100 denotes higher prepayments on the loan and viceversa.
The formula to calculate the loss will be Outstanding * CPR * (Prepay Multiplier/100)
Bloomber vectors are accepted
