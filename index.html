CLO Model Documentation

This project integrates a .Net interface for the cash flow modelling tool for CLOs.

Dependencies

The project is based on the Qt Libraries version 4.8 and the .Net famework 4.0.

In addition to those base libraries it makes use of:

  • The Bloomberg Api v3 (A Bloomberg login is required only if you wish to use Bloomberg functionalities)
  • zlib to manage compressed files

Linked libraries

  • blpapi3_32.dll
  • QtCore4.dll
  • QtGui4.dll
  • zlib1.dll
