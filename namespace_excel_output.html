Sends the information of the loans cash flows to excel to be charted.

Parameters
sourceThe loans cash flows that should be printed
DestinationSheetThe sheet name where the chart is
DestinationIndexThe index of the chart object in the DestinationSheet sheet where the results should be plotted to
PrintDatesDetermines if the dates of the cash flows should be printed
PrintOutstandingDetermines if the tranche outstanding through time should be printed
PrintInterestDetermines if the interest flows should be printed
PrintScheduledDetermines if the scheduled principal flows should be printed
PrintPrepayDetermines if the prepayments flows should be printed
PrintTotalPrincipalDetermines if the total principal (scheduled+prepayments) flows should be printed
PrintLossDetermines if the losses on principal through time should be printed
Returns
S_OK if the output was successful or an error code

This functions sends the informations regarding a tranche cash flows to excel to be plotted in a chart

It will call the VBA function PlotMortgagesFlows passing it the following arguments:

  1. A String containing DestinationSheet
  2. A Long containing DestinationIndex
  3. A Variant containing the dates that will be used as X values in the chart
  4. A Variant containing the total loans amount outstanding through time. It will be printed as an area chart
  5. A Variant containing the loans interest cash flows. It will be printed as a bar chart
  6. A Variant containing the loans scheduled principal cash flows. It will be printed as a bar chart
  7. A Variant containing the loans prepayments cash flows. It will be printed as a bar chart
  8. A Variant containing the loans total principal (scheduled+prepayments) cash flows. It will be printed as a bar chart
  9. A Variant containing the loans loss on principal for the period through time. It will be printed as a bar chart

If a boolean parameter is set to false the relevant array will contain empty values
