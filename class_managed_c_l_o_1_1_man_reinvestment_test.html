Detailed Description

Class defining the reinvestment test.

This class defines the details of reinvestments and the reinvestment test.

Note
The details of the reinvestment bond are used not only by the reinvestment test but also by reinvestment of principal

Define the characteristics of the bond bought using the reinvestment funds.

  • IntrVec A Bloomberg vector defining the coupon or the margin paid by the bond
  • CPRVec The CPR vector used for the reinvestment bond
  • CDRVec The CDR vector used for the reinvestment bond
  • LSVec The LS vector used for the reinvestment bond
  • WALval The WAL vector used for the reinvestment bond
  • PayFreq The number of months between two interest payments from the bond
  • AnnuityVec A Bloomberg-style vector defining whether the bond is an annuity.
    The vector can assume the value N for interest only and Y for annuity and can only move in steps.
    Examples of valid vectors:
    Y
    N 24S Y
    Examples of invalid vectors:
    5 24S N
    N 24R Y
  • FloatingBase the base rate value to be added to IntrVec if it's a margin vector. Use 0 for fixed rate.

Define the characteristics of the reinvestment test.

  • ReinvPer The date on which the reinvestment period will end
  • TstLvl The minimum level of the test for it to pass in a scalar form (1=100%, 1.1=110%)
  • IIshare Share toward acquiring new collateral during reinvestment period in a scalar form (0.5=50%)
  • IRshare Share redemption of notes during reinvestment period in a scalar form (0.5=50%)
  • OIshare Share toward acquiring new collateral after reinvestment period in a scalar form (0.5=50%)
  • ORshare Share redemption of notes after reinvestment period in a scalar form (0.5=50%)
